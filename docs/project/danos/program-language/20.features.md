## 名称无关

变量名、类型名、函数名不是必须的，实际均通过其**唯一标识**做引用，
可在任意时刻以任意字符命名变量、类型和函数。名称只是为代码阅读者而提供的，
便于理解代码逻辑，其对代码编译和执行是无意义的。

## 函数即对象

函数的执行逻辑为AST树形结构，同时包含参数列表、唯一标识、返回类型、
私有性、说明文档、使用样例、修订版本、创建人、所在包（模块）等信息。
